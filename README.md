## Mudmouth

Mudmouth is a network diagnostic library for capturing requests securely on iOS.

### Usage

- Capturing HTTP requests
- Capturing HTTPS requests with MitM

### Requirements

- iOS 16.x
- Xcode 16.x
- fastlane
